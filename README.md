# DerivativeTime是用于管理多个事项以及每个事项的时间的学习办公类软件，它具有快捷方便的特点。
# 开发平台：visual studio

项目需求：
1. 实现事项的添加、修改、删除以及管理不同事项之间的从属关系；
2. 实现事项文件的存储、读取、导出；
3. 实现时间提醒、快速命令、自定义窗口样式等功能；
4. 实现快捷键调出以及快速修改事项功能；
   
技术应用：
1. C#
2. WPF  框架
   
技术重点：
1. 委托与事件的调用
2. 文件输入输出
3. WPF  控件及其依赖属性的绑定
4. WPF  动画应用以及缓动函数的使用
5. Socket  与进程间通信的应用
6. 全局快捷键的实现
